# Telegram Shop Bot

A Telegram bot that lets users browse food, drinks, and desserts, add items to a cart, and checkout with a formatted receipt.

## Features
- Browse categories: Foods, Drinks, Desserts
- View item details with photos and descriptions
- Add/remove items in cart
- Checkout with PDF receipt generation
- Continue ordering after checkout

## Setup
1. Install Python 3.11+
2. Clone the repo
3. Install dependencies: `pip install -r requirements.txt`
4. Create a `.env` file with your bot token and DB credentials
5. Start MySQL server
6. Run the bot: `python app.py`
